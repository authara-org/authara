# Examples

Looking for integration examples?

👉 https://github.com/authara-org/authara-examples

This repository contains minimal example applications showing how to integrate Authara with different technology stacks.

## Available examples

- Go + HTMX
- Go + React
- Node + React (coming soon)

Each example includes:
- a working application
- Docker setup
- Authara integration end-to-end

## When to use examples

Use the examples if you:
- want to see a full working setup
- are integrating Authara into an existing app
- prefer learning by reading code instead of docs
